# Control por planitud diferencial — Doble péndulo invertido

Diseño y simulación de un controlador basado en planitud diferencial para un sistema de doble péndulo invertido sobre un carro. El modelo no lineal se deduce mediante Euler–Lagrange, se linealiza en torno al equilibrio invertido y se obtiene la salida plana a partir de la matriz de controlabilidad para plantear un control de seguimiento de trayectorias.
